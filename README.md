# Covidcore API

## Install requirements:
 ```sh
$ python setup.py develop
```

## Instalation
 ```sh
$ flask db init
$ flask db migrate
$ flask db upgrade
```

### Documentation
In order to make the documentation work you will need to follow these steps:

 ```sh
$ cd app/static
$ npm install
```

## Testing
 ```sh
$ python setup.py test
```
